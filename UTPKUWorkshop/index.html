First Joint Workshop on Bidirectional Transformation in Architecture-Based Component Composition

December 7, 2007.
Room 1621. Science Building #1, Peking University, Beijing, China

Objective

This informal workshop focuses on generative approach to software development, aiming to provide a forum for discussing bidirectional transformation technology in software engineering in general, and bidirectional model synchronization in the Architecture-based Component Composition (ABC) method in specific. It is jointly organized by Key Laboratory of High Confidence Software Technologies of Peking University and Information Processing Laboratory of University of Tokyo.

Topics

Talks are invited on, but not limited to, the following topics:

  • Automatic Software Development
  • Architecture-based Component Composition
  • Bidirectional Transformation Languages
  • Model-driven Software Development
  • Model Transformation/Synchronization
  • Round-trip Software Development
  • Reverse Engineering
  • Software/Program Refinement
  • Software Reuse
