新闻(更多)

  • 2021年7月1日 Probabilistic Delta Debugging被评为ACM SIGSOFT杰出论文。感谢评委的认可。
  • 2021年6月15日 Interactive Patch Filtering as Debugging Aid被ICSME'21接收. 之前工作一直认为修复工具如果不能达到很高的正确率是没用的，这篇工作通过200轮人工调试的实验证明了，如果在程序员复查补丁的时候给予适当的交互式工具辅助，正确率低的修复工具也能帮助程序员，从而为未来修复研究的发展带来了全新空间，比如在无需担心正确率的情况下进一步提升召回率。该论文是我们PLDI'21论文的后续工作。
  • 2021年5月21日 两篇论文被ESEC/FSE'21接收。Probabilistic Delta Debugging提出了首个基于概率建模的差异化调试方法，效果显著超越经典ddmin方法。A Syntax-Guided Edit Decoder for Neural Program Repair提出了首个效果超越传统方法的神经网络程序修复方法。
  • 2020年12月1日 我获得了NASAC青年软件创新奖和ESEC/FSE 2020的杰出审稿人称号。感谢同行的提名和评审委员会的认可。
  • 2020年10月1日 Guiding Dynamic Programming via Structural Probability for Accelerating Programming by Example被OOPSLA'20接收。该工作是我们采用概率来指导基于枚举的程序合成的后续工作，通过用概率指导Flashfill风格的动态规划程序合成，提速达到4-2000倍。
